# 税理士事務所ページのデザイン改善

## 概要

税理士事務所のアーカイブページと詳細ページを、ビジネスライクでおしゃれなデザインに改善しました。

## デザインコンセプト

**「プロフェッショナル × モダン」**

- **カラースキーム**: ブルーグラデーション（#1e3a8a → #3b82f6）を基調
- **トーン**: 信頼感と洗練さを両立
- **UI**: カードデザインとホバーエフェクトで直感的な操作性

## 主な改善点

### 1. アーカイブページ（一覧ページ）

#### ヘッダーセクション
- **背景**: ブルーグラデーション
- **タイトル**: 大きく読みやすいフォント（2.5rem）
- **説明文**: 白色で視認性を確保

#### フィルターセクション
- **背景**: 白色カード
- **ボーダー**: 柔らかい角丸（12px）
- **シャドウ**: 軽いドロップシャドウで浮き上がり効果
- **フォーカス**: ブルーのアウトラインで選択状態を明示

#### 事務所カード
- **レイアウト**: グリッドレイアウト（320px最小幅）
- **ホバーエフェクト**: 
  - 上に4px浮き上がる
  - シャドウが濃くなる
  - ボーダーがブルーに変化
- **サムネイル**: デフォルトで🏛️アイコンを表示
- **タグ**: ブルー系の背景で統一感

### 2. 詳細ページ

#### ヘッダーセクション
- **タイトル**: 大きく目立つ（2.2rem）
- **ボーダー**: 下部に3pxのブルーライン
- **メタ情報**: 都道府県リンクをブルーで強調

#### 得意分野・得意業種セクション
- **レイアウト**: グリッドレイアウト（200px最小幅）
- **タグデザイン**: 
  - グラデーション背景（#eff6ff → #dbeafe）
  - ブルーボーダー
  - ホバーで上に2px浮き上がる

#### 事務所情報テーブル
- **ヘッダー**: ダークブルー背景（#1e3a8a）
- **行**: 交互に背景色を変えて読みやすく
- **リンク**: ブルーで統一

#### お問い合わせセクション
- **背景**: ライトブルーグラデーション
- **ボタン**: 
  - 公式サイト: ブルーグラデーション（メインCTA）
  - 電話: 白背景 + ダークブルーボーダー
  - メール: 白背景 + ブルーボーダー
- **ホバーエフェクト**: 浮き上がり + シャドウ

## レスポンシブ対応

### ブレークポイント: 768px

**モバイル表示時の調整:**
- ヘッダータイトルを1.8remに縮小
- フィルターグリッドを1カラムに変更
- カードを1カラムに変更
- テーブルのフォントサイズを縮小
- お問い合わせボタンを縦並びに変更

## 技術仕様

### CSS構造
```
税理士事務所ページ
├── アーカイブページ (.tax-office-archive)
│   ├── ヘッダー (.archive-header)
│   ├── フィルター (.service-filter)
│   ├── カード一覧 (.service-cards)
│   └── ページネーション (.pagination)
└── 詳細ページ (.tax-office-single)
    ├── ヘッダー (.tax-office-single__header)
    ├── サムネイル (.tax-office-single__thumbnail)
    ├── 得意分野 (.tax-office-single__services)
    ├── 得意業種 (.tax-office-single__industries)
    ├── 事務所情報 (.tax-office-single__info)
    └── お問い合わせ (.tax-office-single__contact)
```

### トランジション
- すべてのホバーエフェクト: `transition: all 0.3s`
- スムーズな視覚フィードバック

### シャドウ
- カード通常時: `0 2px 8px rgba(0, 0, 0, 0.08)`
- カードホバー時: `0 8px 24px rgba(0, 0, 0, 0.12)`
- ボタンホバー時: `0 4px 12px rgba(59, 130, 246, 0.3)`
